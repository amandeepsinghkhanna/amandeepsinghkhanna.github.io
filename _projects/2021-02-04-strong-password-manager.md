This is a flask application for generating strong passwords. It generates a password that is 16 characters long, contains at least one alphabet in uppercase, at least one character in lowercase, at least one numeric character and at least one special character. You can find a running version of the application by clicking <a href="http://adamsandler.pythonanywhere.com/">here</a>.
